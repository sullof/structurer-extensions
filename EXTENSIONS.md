# Structurer Extensions

This page lists curated extension structures.

## How to use these extensions

1. Download the JSON file of the extension you want from this repository.
2. Open Structurer and go to Dashboard `... Actions`.
3. Click `Import custom structure`.
4. Select the downloaded JSON file.
5. Once imported, the new structure is available when creating a story.

If you want to contribute a new extension, please follow the contribution guide: [CONTRIBUTING_EXTENSIONS.md](CONTRIBUTING_EXTENSIONS.md).

## English

- [Freytag's Pyramid](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/freytag-pyramid.json) - Classic dramatic arc with exposition, climax, and denouement.
- [Fichtean Curve](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/fichtean-curve.json) - Crisis-driven structure focused on escalating tension.
- [Snowflake Method (Outline Phases)](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/snowflake-method-outline.json) - Practical planning steps for expanding an idea into a full outline.
- [Holly Lisle's Clinic Method](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/holly-lisle-clinic.json) - Revision-oriented structure for diagnosing and fixing story weaknesses.
- [W-Plot / W-Method](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/w-plot-method.json) - Alternating downturn/recovery rhythm often used for pace control.
- [Mini-Movie Method](https://raw.githubusercontent.com/sullof/structurer-extensions/main/extensions/structures/en/mini-movie-method.json) - High-clarity beat sequence often used in commercial/YA fiction planning.
